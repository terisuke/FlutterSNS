# 卒業制作(仮)

-FirebaseとFlutterを使ったTwitterのようなテキスト型SNSです。
Algolia<https://www.algolia.com/>を搭載し検索機能を、
amazoncomprehend<https://aws.amazon.com/jp/comprehend/>を搭載し、ツイートの内容がどんな感情なのかを読み取れるようにしています。
